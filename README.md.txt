# CNN Spam Classifier

A Convolutional Neural Network (CNN)-based spam message classifier using TensorFlow/Keras and a public SMS Spam dataset.

##  Dataset
Uses the [UCI SMS Spam Collection Dataset](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset) with over 5,000 SMS messages labeled as spam or ham.

## Usage
- Open `notebooks/cnn_spam.ipynb` in Google Colab
- Train the model
- Save the trained CNN model

## Requirements
Install dependencies with:
```bash
pip install -r requirements.txt
```
